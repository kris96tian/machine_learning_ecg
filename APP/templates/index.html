ECG Diagnose Prediction App (Silent Heart Attacks)

This app allows you to analyze ECG data using a pre-trained AI model. Follow these steps to use the tool:

  1. Select your ECG file (Currently supported formats include CSV and HDF5).
  2. Choose the file type from the dropdown menu.
  3. Set the desired target length for your data. The tool will automatically adjust your ECG data to this length.
  4. Click "Predict" to analyze the ECG data.

Note: The model was trained on data from 3,750 patients, with each ECG sample having a shape of (5300, 12), representing 5300 data points across 12 ECG leads.
